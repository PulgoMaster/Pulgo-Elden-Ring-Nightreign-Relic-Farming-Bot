# Code of Conduct

This project adopts the **[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)** as its code of conduct.

The full text is available at the link above and applies to all project spaces (this repository, issues, discussions, pull requests, and any project-related communication channels).

## Reporting

If you observe behavior that violates the Contributor Covenant in any project space, report it privately to the maintainer by sending a direct message to [`@PulgoMaster`](https://github.com/PulgoMaster) on GitHub.

Reports will be reviewed privately. The maintainer will determine and carry out appropriate response actions per the Contributor Covenant enforcement guidelines.
